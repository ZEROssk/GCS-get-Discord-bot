### Get Google Calender Schedule Discord Bot
#### command
```
!man, !today, !week, !nweek
```
| Command | Function                          |
|:-------:|:---------------------------------:|
| !man    | Show command all                  |
| !today  | Today schedule                    |
| !week   | Get from today to friday schedule |
| !nweek  | Get next week schedule            |
